# OrcunGuler1.github.io

Developer page and legal pages for Orcun Guler mobile games.

## Live URLs
- Home: `https://orcunguler1.github.io/`
- Privacy policies: `https://orcunguler1.github.io/privacy/`
- Terms of Service: `https://orcunguler1.github.io/terms/`
- AdMob crawler file: `https://orcunguler1.github.io/app-ads.txt`

## Structure
- `index.html`: Developer landing page and game cards
- `privacy/index.html`: Single privacy page with app anchors
- `terms/index.html`: Single terms page with app anchors
- `styles.css`: Shared visual styles
- `app-ads.txt`: AdMob declaration file

## Update checklist
1. Add or update a game card in `index.html`.
2. Add or update the matching section in `privacy/index.html`.
3. Add or update the matching section in `terms/index.html`.
4. Keep anchor IDs stable (`#circuit-connect`, `#echomaze`, `#oopsallmines`, `#seconds-to-spare`).
5. If publisher details change, update `app-ads.txt`.
6. Commit and push to `main` to trigger GitHub Pages deployment.
